# 🌦️ Weather Info Checker (Python + Termux)

A simple command-line Python tool to check current weather info for any city using OpenWeatherMap API. Designed to run perfectly in **Termux** or any terminal.

---

## ✅ Features

- 🔍 Check weather by city name  
- 🌡️ Shows temperature, condition, humidity  
- 🌈 Colorful output in terminal  
- 🐍 Pure Python, runs on Termux
